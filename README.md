# Strategic Product Placement Analysis

This folder arranges the completed Data Analytics project in a SmartBridge-style submission format.

## Structure

- `Assignments/` - reference and assignment material.
- `Project Design & Planning/` - ideation, design, architecture, and planning phase notes.
- `Project Development Phase/` - sprint-wise development summary.
- `Final Deliverables/` - final source code, report, Tableau workbook, dataset, demo link, and documentation.

## Project

Strategic Product Placement Analysis is a Flask and Tableau analytics project for evaluating how FMCG product placement affects retail sales volume.

The original runnable project remains in the main `dataanalyis` folder. This folder is an arranged submission copy.
