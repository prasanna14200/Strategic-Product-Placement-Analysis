About This Project

Understanding the strategic analysis behind product placement optimization in the FMCG retail sector.

Project Overview

The Strategic Product Placement Analysis project investigates how product positioning within retail environments impacts sales performance in the Fast-Moving Consumer Goods (FMCG) sector.

Using a comprehensive dataset from Kaggle, this project performs in-depth exploratory data analysis, creates interactive Tableau visualizations, and deploys a Flask web application to present findings in an accessible format.

The analysis covers multiple dimensions including product position (Front of Store, End-cap, Aisle), pricing strategies, promotional campaigns, foot traffic patterns, consumer demographics, and seasonal effects on sales volume.

Methodology

Our Approach

1

Data Collection

Downloaded the product positioning dataset from Kaggle containing 500+ records with 9 key variables.

2

Data Cleaning

Handled missing values, removed duplicates, standardized formats, validated types, and capped outliers.

3

Analysis & Viz

Performed EDA with Python and created 8+ interactive Tableau visualizations with calculated fields.

4

Deployment

Built a Flask web app embedding the Tableau dashboard and story, published to Tableau Public.
