# Strategic Product Placement Analysis

A full-stack data analytics web application using Flask and Tableau to evaluate the impact of FMCG product placement strategies on retail sales volume. Built for the **SmartBridge Externship — Data Analytics with Tableau**.

**Topics:** tableau, flask, data-analytics, fmcg, dashboard

## Overview

This project downloads and cleans the [Kaggle dataset](https://www.kaggle.com/datasets/amitvkulkarni/impact-of-product-positioning-on-sales), performs exploratory data analysis in Python, builds interactive Tableau visualizations (dashboard + 3-scene story), and deploys a Flask web application embedding the published Tableau views.

## Key Features

- Automated data cleaning pipeline (missing values, duplicates, outliers, standardization)
- 8+ Tableau visualizations with calculated fields and interactive filters
- Interactive dashboard with 5 filters (Category, Position, Promotion, Season, Demographics)
- 3-scene Tableau story (Business Overview → Position Analysis → Recommendations)
- Flask web app with Home, Dashboard, Story, and About pages
- REST API endpoints for KPIs and dataset summary

## Quick Start

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Download dataset (requires Kaggle API credentials)
python scripts/download_dataset.py

# 3. Clean data
python scripts/data_cleaning.py

# 4. Run analysis
python scripts/data_analysis.py

# 5. Create Tableau Hyper extract (official Hyper API — data only)
python scripts/create_tableau_hyper.py

# 6. Build workbook in Tableau Desktop 2026 — see tableau/Workbook_Build_Guide.md

# 7. Start Flask app
python app.py
```

Open http://127.0.0.1:5000 in your browser.

## Tableau Workbook

> **Important:** Workbooks (`.twb`/`.twbx`) are **not** generated by Python scripts. Hand-written workbook XML is invalid. Build the workbook in **Tableau Desktop / Public Edition 2026**.

1. Run `python scripts/create_tableau_hyper.py` to create `tableau/data/product_positioning_sales.hyper`
2. Open Tableau Public Edition 2026 and connect to the Hyper file
3. Follow `tableau/Workbook_Build_Guide.md` to build dashboard + story
4. Publish using `tableau/Tableau_Public_Publishing_Guide.md`
5. Update `TABLEAU_DASHBOARD_URL` and `TABLEAU_STORY_URL` in `app.py` or `.env`
