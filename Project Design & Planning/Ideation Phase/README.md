# Ideation Phase

## Problem Statement

Retail and FMCG teams need to understand how product placement, pricing, promotions, seasonality, and demographics affect sales volume.

## Proposed Solution

Create a data analytics application that cleans the dataset, identifies sales patterns, builds Tableau visualizations, and presents insights through a Flask web app.

## Target Users

- Retail managers
- FMCG strategy teams
- Business analysts
- Store operations teams

## Expected Output

The project should reveal the most effective product positions, high-performing categories, demographic patterns, and promotion impact.
